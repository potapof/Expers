## Формула баланса «текст ↔ изображение»

При рендеринге на странице (ширина контента ~720px, шрифт 16px/1.6):

| Дизайн      | Изображение | Высота изобр. | ≈ символов текста             |
| ----------- | ----------- | ------------- | ----------------------------- |
| image-only  | 1200×600    | 600px         | 100-200 (подпись)             |
| image-right | 800×400     | 400px         | 1400-1600                     |
| image-left  | 800×400     | 400px         | 1400-1600                     |
| text-only   | —           | —             | 2500-3500                     |
| table       | —           | —             | 1500-2500 (без самой таблицы) |

**Правило для image-right / image-left:**
Текстовый блок рядом с изображением должен быть **1400-1600 символов**,
чтобы визуально занимать ту же высоту, что и изображение 800×400 (400px).

**Формула в промпте:**

```
800x400 картинка = 400px высоты ≈ 1500 символов текста
1200x600 картинка = 600px — image-only, подпись 100-200 символов
text-only блоки: 2500-3500 символов
```

| Поле         | Мин | Макс | Описание                          |

## Риски

| Риск                                             | Решение                                                                        |
| ------------------------------------------------ | ------------------------------------------------------------------------------ |
| ИИ игнорирует лимиты символов на секцию          | Warning вместо error. Пользователь доработает в визарде                        |
| ИИ путает порядок секций                         | `sectionNumber` в ключах: `section1Заголовок` → однозначный порядок            |
| 11 итераций — пользователь устаёт                | Сохранить паузу «Проверьте и напишите Продолжить». За один сеанс ≈ 30-45 минут |
| Разбиение итерации 8 на 4 — больше ручной работы | Итерации 8-11 короче (по 2-3 поля), компенсируется скоростью                   |
| ИИ не справляется с самопроверкой в итерации 11  | Парсер делает свою проверку независимо. Самопроверка ИИ — дополнительный слой  |
