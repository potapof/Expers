# Expers.ru — Экспертная платформа для бизнеса

Открытая экспертная платформа (аналог Medium по дизайну и функционалу), где читатели бесплатно получают доступ к GEO-оптимизированным экспертным статьям, а эксперты публикуют контент через 12-шаговый мастер заполнения, обеспечивающий максимальную адаптацию к требованиям ИИ-поисковиков (Generative Engine Optimization).

## Основные возможности

- **Каталог статей** — лента статей, структурированная по 13 отраслям бизнеса, с боковым меню для фильтрации по отраслям, подразделам и категориям
- **GEO-разметка** — каждая статья содержит полный набор GEO-блоков (TL;DR, Key Facts, FAQ, HowTo, Featured Snippet и др.) со schema.org JSON-LD разметкой
- **12-шаговый мастер публикации** — конструктор статей из карточек-разделов с выбором дизайна, загрузкой изображений, встроенным редактором текста и предпросмотром
- **Личный кабинет читателя** — персонализированная лента, избранное, комментарии, управление интересами
- **Аутентификация** — регистрация и вход экспертов с JWT

## Стек технологий

- **Фреймворк:** Next.js 16 (SSR/App Router), React 19, TypeScript
- **Стилизация:** Tailwind CSS 4, shadcn/ui, tw-animate-css
- **Иконки:** lucide-react
- **Baлидация:** zod
- **Уведомления:** sonner
- **База данных:** DynamoDB (Yandex Cloud Document API)
- **Аутентификация:** bcryptjs, jsonwebtoken
- **Инфраструктура:** Docker Compose (dynamodb-local для разработки)

## Запуск

### Требования

- Node.js 20+
- Docker и Docker Compose

### Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск инфраструктуры (DynamoDB local) и приложения
docker compose up -d

# Применение миграций БД
docker compose exec app npm run db:migrate

# Приложение доступно на http://localhost:8080
```

### Команды

| Действие                 | Команда                                      |
| ------------------------ | -------------------------------------------- |
| Запустить dev-сервер     | `docker compose up -d`                       |
| Перезапустить приложение | `docker compose up -d --force-recreate app`  |

| Линтинг                  | `npm run lint`                               |
